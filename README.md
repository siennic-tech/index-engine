# index-engine
Index Engine is a non-profit Siennic project intended on connecting the best parts of our lives.

In 1997, the Siennic government had one goal: Create the internet, better than ever, just for Siennics. So, headquartered in Fort City, Fortlund, they started work. They built a frontend that could import and read discs containing the entire web, but it was very problematic. It couldnt be effectively updated, and had to store, or at least run, hundreds of thousands of pages and documents locally. 

So to solve this problem, FUIDR (Fortlund Institute of Internet Development and Research) has developed this project.
